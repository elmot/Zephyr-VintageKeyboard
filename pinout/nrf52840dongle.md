The dongle drawing
===

![nrf52840dongle](nrf52840dongle.png)

i2c pins:
---

* SDA - P0.29
* SCL - P0.31

SX1509B interrupt
---
P0.02

LEDs
---

* On-Board:
  * LD1	    Green	P0.06
  * LD2R	Red	    P0.08
  * LD2G	Green	P1.09
  * LD2B	Blue	P0.12
* External:

  | LED   | Pin   | Function            |
  |-------|-------|---------------------|
  | LED1  | P0.13 | CAPS LOCK           |
  | LED2  | P0.15 |                     |
  | LED3  | P0.17 |                     |
  | LED4  | P0.20 | POWER ON            |
  | LED4  | P0.22 | BLUETOOTH CONNECTED |
  | LED4  | P0.24 | USB CONNECTED       |
Button
---
P1.06
